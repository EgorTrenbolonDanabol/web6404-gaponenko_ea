# Лабораторная работа 3 - Core Javascript

## Как пользоваться

Для того, чтобы всё корректно работало, у вас должен быть установлен [node](https://nodejs.org/en/) и [git](https://git-scm.com/)

## Описание

Набор базовых заданий на JavaScript для тренировки работы с числами, циклами, рекурсией и объектами.
Тестирование выполняется с помощью Mocha и встроенного модуля assert.

Установка и запуск
npm install
npm test

Файл с задачами: src/core.js
Файл с тестами: src/core.test.js

# Лабораторная работа 4 — Modern CSS и OOP JavaScript

## Описание
модернизировать web-сайта из лабораторной работы №2  с использованием Flexbox и Grid для вёрстки, CSS-переменных для цветовой схемы и анимации. # Лабораторная работа №5

## Описание
В рамках лабораторной работы №5 был модернизирован web‑сайт из предыдущей лабораторной работы с использованием современных возможностей HTML, CSS и JavaScript. Реализована динамическая валидация формы на основе событийной модели, добавлены текстовые подсказки для пользователя, а отправка данных осуществляется через POST‑запрос после успешной проверки. Для одной из секций сайта настроено периодическое асинхронное получение данных с сервера с обработкой возможных ошибок, при этом в качестве серверной части используется mock‑JSON‑server.
